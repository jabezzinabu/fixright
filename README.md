## 1. Supabase Setup

### Create the estimates table
Run this SQL in your Supabase project → SQL Editor:

```sql
create table estimates (
  id uuid default gen_random_uuid() primary key,
  user_id text not null,
  title text,
  price_range text,
  difficulty text,
  timeline text,
  region text,
  approach text,
  description text,
  materials jsonb,
  steps jsonb,
  warnings jsonb,
  tip text,
  created_at timestamp with time zone default now()
);

-- Enable Row Level Security
alter table estimates enable row level security;

-- Allow anyone with a matching user_id to read/write their own estimates
create policy "Users can manage their own estimates"
  on estimates for all
  using (user_id = current_setting('request.headers')::json->>'x-user-id');
```

### Add your credentials
Open `www/index.html` and replace these two lines near the top of the `<script>` section:
```js
const SUPABASE_URL = 'YOUR_SUPABASE_URL';
const SUPABASE_ANON_KEY = 'YOUR_SUPABASE_ANON_KEY';
```

## 2. Capacitor / Mobile Setup

### Prerequisites
- Node.js 18+
- For iOS: Mac with Xcode 15+ installed
- For Android: Android Studio (any OS)

### Install dependencies
```bash
cd fixright
npm install
```

### Add platforms (first time only)
```bash
npx cap add ios
npx cap add android
```

### Sync web app to native projects
Run this every time you change index.html:
```bash
npx cap sync
```

### Open in Xcode (iOS)
```bash
npm run open:ios
# Then in Xcode: select a simulator or device → press ▶
```

### Open in Android Studio
```bash
npm run open:android
# Then in Android Studio: Run → Run 'app'
```

## 3. App Store Submission Checklist

### iOS (Apple)
- [ ] Apple Developer account ($99/year) at developer.apple.com
- [ ] Set Bundle ID in Xcode to match `com.fixright.app`
- [ ] Add app icons (1024×1024 PNG) via Xcode asset catalog
- [ ] Add launch screen
- [ ] Set camera usage description in Info.plist:
  `NSCameraUsageDescription` = "FixRight uses your camera to photograph the repair area for a more accurate estimate."
  `NSPhotoLibraryUsageDescription` = "FixRight accesses your photos to analyse your repair project."
- [ ] Archive → Distribute via TestFlight first

### Android (Google Play)
- [ ] Google Play Developer account ($25 one-time) at play.google.com/console
- [ ] Set applicationId in `android/app/build.gradle` to `com.fixright.app`
- [ ] Add app icons via Android Studio resource manager
- [ ] Generate signed APK/AAB for release
- [ ] Add camera permission to AndroidManifest.xml (Capacitor does this automatically)

---

## 4. Updating the App

1. Edit `www/index.html`
2. Run `npx cap sync`
3. Rebuild in Xcode / Android Studio

That's it — no native code changes needed for web-only updates.
